# QUIC_Upload

### Server
```
$ go run ./server/server_upload.go
```
Parameters:  
- `-h`: host  
- `-p`: port  
- `-d`: path to store receiving files  
- `-l`: path to store log files  

### Client
```
$ go run ./client/client_upload.go
```
Parameters:  
- `-h`: host  
- `-p`: port  
- `-f`: transfering file  
- `-l`: path to store log files
